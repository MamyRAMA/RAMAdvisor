OBJECTIF : PRESERVATION DU CAPITAL
Description : Priorité absolue à la protection du capital contre les pertes. La croissance est un objectif très secondaire. Adapté aux investisseurs ayant une aversion totale au risque ou des besoins de liquidité imminents.
Typologie : PRESERVATION-1Y

Objectif : Préservation du capital

Profil de Risque : Très faible

Horizon de Temps : 1 an

Liquidité Requise : Très élevée

Allocation Modèle :

Liquidités / Fonds Monétaires : 70%

Obligations d'État (court terme < 2 ans) : 30%

Actions : 0%

Immobilier : 0%

Actifs Alternatifs : 0%

Typologie : PRESERVATION-3Y

Objectif : Préservation du capital

Profil de Risque : Très faible

Horizon de Temps : 3 ans

Liquidité Requise : Élevée

Allocation Modèle :

Liquidités / Fonds Monétaires : 40%

Obligations d'État (court terme) : 50%

Obligations d'Entreprise (haute qualité) : 10%

Actions : 0%

Immobilier : 0%

Actifs Alternatifs : 0%

Typologie : PRESERVATION-5Y

Objectif : Préservation du capital avec une légère appréciation

Profil de Risque : Faible

Horizon de Temps : 5 ans

Liquidité Requise : Modérée à élevée

Allocation Modèle :

Liquidités / Fonds Monétaires : 20%

Obligations d'État : 40%

Obligations d'Entreprise (haute qualité) : 25%

Actions (grandes capitalisations, faible volatilité) : 15%

Immobilier : 0%

Actifs Alternatifs : 0%

OBJECTIF : CROISSANCE AGRESSIVE / SPÉCULATIF

Profil de Risque : Très élevé / Spéculatif

NOTE ADDITIONNELLE SUR LES HORIZONS COURTS POUR PROFILS DYNAMIQUES
Les profils dynamiques avec des horizons courts (1-3 ans) sont atypiques et risqués. Ils correspondent à des stratégies opportunistes ou spéculatives.
Typologie : OPPORTUNISTE-1Y

Objectif : Gain rapide spéculatif

Profil de Risque : Extrêmement élevé

Horizon de Temps : 1 an

Liquidité Requise : Faible (le capital est à risque)

Allocation Modèle :

Liquidités / Fonds Monétaires : 20%

Actions (Thématiques, forte volatilité) : 60%

Actifs Alternatifs (Crypto-actifs) : 20%
